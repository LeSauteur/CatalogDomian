# Прототип каталога недвижимости

Это локальный демонстрационный прототип сайта-каталога объектов недвижимости. Он собран на обычных HTML, CSS и Vanilla JavaScript, без backend, авторизации, CRM, базы данных и сборщика.

## Как открыть

Откройте `_prototype_catalog/index.html` в браузере.

Или запустите локальный сервер из корня проекта:

```bash
python -m http.server 8000
```

и откройте:

```text
http://localhost:8000/_prototype_catalog/
```

## Страницы

- Главная: `index.html`
- Каталог: `catalog.html`
- Объект: `object.html?id=apt-001`
- Добавление объекта: `admin.html`

## Как проверить сценарий

1. Откройте `admin.html`.
2. Заполните объект.
3. Нажмите `Опубликовать`.
4. Перейдите в `catalog.html`.
5. Новый объект появится в каталоге.
6. Откройте карточку через кнопку `Подробнее`.

## Как работает добавление

Форма сохраняет опубликованные объекты в `localStorage` под ключом `domianPrototypeObjects`. Черновик тоже сохраняется, но не показывается в каталоге. Для демо можно загрузить фото через файл, вставить URL или выбрать одно из быстрых демо-фото.

## Что это не делает

Это не production-CMS и не публичная версия сайта. Здесь нет серверной загрузки фото, модерации, ролей, интеграции с CRM и настоящей базы данных.

Для production `localStorage` можно заменить на CMS, backend или headless-хранилище, а форму подключить к CRM, Web3Forms или собственной админке.
